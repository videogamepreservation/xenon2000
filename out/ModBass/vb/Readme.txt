Requirements
============
The BASS VB API requires Visual Basic 4-32 bit or above due to 
the use of callbacks etc.

Things To Remember
==================
When using the BASS functions, wherever you need to use TRUE or FALSE
you must use BASSTRUE or BASSFALSE for the BASS functions to recognise
these boolean values.
